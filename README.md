# CS210-Programming-Languages

The Corner Grocer Item-Tracking Program is designed to analyze a text file containing records of purchased items, tracking the frequency of each item and displaying the results.

I successfully planned the coding process and outlined the program’s requirements. However, I could have enhanced the code by incorporating multiple programming languages to add more functionality.

The most challenging aspect was writing the code to access data from the workspace. Despite this, utilizing information from a separate file will be highly beneficial in the future.

I ensured that each part of the code was organized separately and included comments to facilitate easy maintenance and readability.
